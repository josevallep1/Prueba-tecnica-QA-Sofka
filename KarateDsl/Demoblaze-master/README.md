# Documentación del Proyecto

## Introducción
Este proyecto es un ejemplo de prueba técnica para demostrar habilidades en QA y automatización de pruebas utilizando Karate DSL.

## Requisitos
- Java 8 o superior
- Maven

## Estructura del Proyecto
- **src/test/java**: contiene los archivos de prueba.
- **src/test/resources**: contiene los archivos de características de Karate.
- **pom.xml**: archivo de configuración de Maven.

## Configuración del Entorno
1. Clonar el repositorio:
   ```bash
   git clone https://github.com/josevallep1/Prueba-tecnica-QA-Sofka.git
   ```
2. Navegar al directorio del proyecto:
   ```bash
   cd Prueba-tecnica-QA-Sofka
   ```
3. Ejecutar las pruebas:
   ```bash
   mvn test
   ```
